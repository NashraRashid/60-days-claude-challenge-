# Day 35 – Prompt Puzzle

## Project
Prompt Puzzle

## Objective

Build an interactive web application that helps users improve their prompt engineering skills through engaging challenges, performance analysis, and real-time feedback.

---

## Features

- Interactive Prompt Puzzle Challenges
- Multiple Domains and Difficulty Levels
- Prompt Performance Report
- Instant Feedback
- Replay with Randomized Scenarios
- Responsive User Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI

---

## What I Learned

- The importance of writing clear and structured prompts.
- How context influences AI-generated responses.
- Different prompt engineering techniques for better results.
- How feedback can help improve prompt quality.
- Building interactive web applications using HTML, CSS, and JavaScript with Claude AI.

---

## Challenges Faced

- Understanding prompt evaluation techniques.
- Designing an engaging challenge flow.
- Managing user interactions and performance tracking.

---

## Outcome

Successfully developed a Prompt Puzzle application that enables users to practice prompt engineering through interactive challenges and performance reports, making AI learning more engaging and practical.
