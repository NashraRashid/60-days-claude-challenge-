# Day 8 – Personal Environmental Health Analyzer 🌍

## Challenge

Build an interactive Environmental Health Analyzer that converts environmental data into personalized health insights.

## Project Overview

The Personal Environmental Health Analyzer is a frontend web application that visualizes environmental conditions and explains their potential impact on daily health.

## Features

* Interactive AQI Dashboard
* PM2.5 & PM10 Analysis
* City Comparison
* Environmental Health Score
* Air & Water Quality Insights
* Health Impact Analysis
* Personalized Recommendations
* Dynamic Charts & Graphs
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* SVG Charts

## Key Learnings

* Creating interactive dashboards
* Building responsive user interfaces
* Visualizing environmental data
* Designing meaningful user experiences
* Using AI to accelerate frontend development

## Outcome

This project demonstrates how environmental data can be transformed into easy-to-understand insights through an interactive dashboard. It strengthened my frontend development skills while showcasing practical data visualization techniques.
