# Day 34 – Marketing Detective

## Project
Marketing Detective

## Objective

Develop an interactive web application that helps users investigate failed marketing campaigns, identify the main marketing mistake, and learn how campaigns can be improved.

---

## Features

- Theme Selection
- Interactive Investigation Board
- Drag-and-Drop Evidence Cards
- Campaign Metrics Dashboard
- Marketing Mistake Detection
- Expert Explanation & Verdict
- Marketing Learning Report
- Randomized Marketing Case Studies
- Responsive User Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI

---

## What I Learned

- How marketing campaigns are analyzed.
- The importance of audience targeting.
- Why budget allocation impacts campaign performance.
- How campaign metrics help identify marketing issues.
- Common marketing mistakes such as:
  - Wrong audience targeting
  - Poor channel selection
  - Overspending on ineffective platforms
  - Weak campaign messaging
- Building interactive web applications using HTML, CSS, and JavaScript.
- Using Claude AI to rapidly generate and improve complete applications.

---

## Challenges Faced

- Understanding different marketing case studies.
- Managing drag-and-drop interactions.
- Organizing multiple investigation screens.
- Displaying campaign data in an interactive way.

---

## Outcome

Successfully developed a Marketing Detective application that allows users to investigate marketing campaigns, analyze evidence, discover marketing mistakes, and learn best practices through an engaging detective-style experience.

---


## Key Takeaway

This project strengthened my understanding of marketing analysis while improving my skills in HTML, CSS, JavaScript, and AI-assisted web development using Claude.
