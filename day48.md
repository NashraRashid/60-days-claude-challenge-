# Day 48 – Coding Bootcamp & Course Comparator 🎓

## Overview

For Day 48 of the **60 Days Claude Challenge**, I built an interactive **Coding Bootcamp & Course Comparator** that helps career-switchers evaluate different learning platforms based on their personal priorities.

Instead of relying on static rankings, the application dynamically updates recommendations as users adjust the importance of different decision criteria.

## Features

* 📊 Interactive weighted scoring system
* 🎯 Adjustable priority sliders
* 🏆 Real-time course rankings
* 📚 Source-backed comparison methodology
* ⚖️ Transparent estimate indicators
* 📱 Responsive user interface

## Platforms Compared

* Coursera Professional Certificates
* Udemy
* freeCodeCamp
* General Assembly Software Engineering Immersive

## Evaluation Criteria

* Cost
* Job Placement & Career Support
* Certification Recognition
* Flexibility

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design

## What I Learned

* Building interactive comparison dashboards
* Implementing weighted scoring algorithms
* Designing decision-support interfaces
* Organizing research into actionable insights
* Improving frontend development using AI-assisted workflows

## Key Takeaway

A great decision-support tool doesn't tell users what to choose—it adapts to their priorities and presents transparent, research-backed comparisons that help them make informed decisions.
