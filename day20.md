# Day 20 – AI Face Puzzle Game

## Challenge Objective

Build an AI-powered Face Puzzle Game using Claude AI that captures a user's photo through the webcam and converts it into an interactive puzzle.

---

## Technologies Used

* HTML
* CSS
* JavaScript
* Claude AI
* Web Camera API
* Local Storage

---

## Features Implemented

### Camera Integration

* Access webcam using browser media devices API
* Capture user photo directly from the browser

### Puzzle Generation

* Convert captured image into puzzle pieces
* Support multiple difficulty levels:

  * 3×3 (Easy)
  * 4×4 (Medium)
  * 5×5 (Hard)

### Game Mechanics

* Shuffle puzzle pieces
* Drag-and-drop functionality
* Touch support for mobile devices
* Puzzle completion detection

### Game Statistics

* Timer tracking
* Move counter
* Completion results

### Leaderboard

* Save scores using Local Storage
* Display best completion times

### User Experience

* Responsive design
* Mobile-friendly interface
* Interactive gameplay

---

## Testing Performed

### Functionality Testing

* Camera permission access
* Photo capture
* Puzzle generation
* Drag-and-drop interactions
* Touch controls
* Timer functionality
* Move counting
* Leaderboard updates

### Browser Testing

* Opened HTML file locally
* Verified responsive layout
* Tested game completion flow

---

## Key Learnings

* How browser camera APIs work
* Using MediaDevices for webcam access
* Image slicing and puzzle generation concepts
* Implementing drag-and-drop interactions
* Managing game state with JavaScript
* Storing game data using Local Storage
* Rapid prototyping with AI-generated code

---

## Project Outcome

Successfully created a browser-based AI Face Puzzle Game that captures a user's image, converts it into puzzle pieces, and provides an engaging interactive gaming experience.
