# Day 39 – PDF Splitter & Merger

## Project

PDF Splitter & Merger

## Objective

Develop a browser-based PDF utility that enables users to split and merge PDF files securely using client-side processing, without uploading documents to external servers.

---

## Features

- PDF Splitter
- PDF Merger
- Drag-and-Drop File Upload
- Live PDF Preview
- Client-Side Processing
- Offline Functionality
- Modern Commercial UI/UX
- Responsive Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI

---

## What I Learned

- How client-side processing enhances privacy and security.
- Creating drag-and-drop workflows for better usability.
- Designing commercial-quality productivity applications.
- Building fully offline browser applications.
- Using Claude AI to generate complete web applications efficiently.

---

## Challenges Faced

- Managing PDF file interactions in the browser.
- Designing an intuitive and responsive interface.
- Organizing file processing workflows for a seamless user experience.

---

## Outcome

Successfully developed a browser-based PDF Splitter & Merger that allows users to manage PDF documents securely with an intuitive interface, live previews, and offline functionality.
