# Day 37 – Task Compass

## Project

Task Compass

## Objective

Develop an interactive web application that helps users understand workplace responsibilities, task delegation, and team collaboration through practical challenges.

---

## Features

- Workplace Type Selection
- Stage 1: Who Owns This?
- Stage 2: Task Routing
- Stage 3: Collaboration Challenge
- Organizational Thinking Dashboard
- Responsive User Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI

---

## What I Learned

- How organizations assign responsibilities to different departments.
- The importance of effective task delegation and routing.
- How teamwork and collaboration improve workplace efficiency.
- Building interactive web applications with HTML, CSS, JavaScript, and Claude AI.

---

## Challenges Faced

- Understanding workplace roles and responsibilities.
- Designing logical task-routing workflows.
- Creating an engaging and interactive user experience.

---

## Outcome

Successfully developed a Task Compass application that simulates workplace task management, helping users improve organizational thinking, collaboration, and decision-making skills.
