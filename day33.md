# Day 33 – Media Integrity Analyzer

## Project
Media Integrity Analyzer

## Objective

Build an interactive web application that helps users identify misleading headlines and emotional manipulation in media content.

---

## Features

- Headline Detective Challenge
- Emotion Detector
- Live Media Integrity Metrics
- Media Integrity Dashboard
- Interactive User Interface
- Responsive Design
- Replay with Randomized Scenarios

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI

---

## What I Learned

- How misleading headlines influence readers.
- How emotional manipulation is used in media.
- The importance of media literacy.
- Building interactive web applications using HTML, CSS, and JavaScript.
- Using Claude AI to generate and improve complete web applications.

---

## Challenges Faced

- Understanding how media manipulation works.
- Organizing multiple interactive screens.
- Managing JavaScript logic for different user interactions.

---

## Outcome

Successfully developed an interactive Media Integrity Analyzer that teaches users how to critically evaluate online content through engaging challenges and a visual dashboard.
