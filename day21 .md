# Day 21 – Digital Privacy Dashboard (Claude AI Challenge)

## Objective

Generate an interactive Digital Privacy Dashboard using Claude AI and analyze the privacy implications of a sample digital footprint dataset.

## Tools Used

* Claude AI
* HTML
* CSS
* JavaScript
* GitHub

## Task Summary

Created a Digital Privacy Dashboard that visualizes:

* Digital Footprint Score
* Privacy Score
* Exposure Heatmap
* Company Exposure Ranking
* Data Collection Matrix
* Risk Radar
* Digital Twin Profile
* Most Valuable Data Assets
* Privacy Improvement Simulator
* Final Privacy Assessment

## Dashboard Findings

### Digital Footprint Score

**72/100**

The score indicates a significant online presence across multiple digital platforms.

### Privacy Score

**44/100**

The privacy score suggests moderate privacy exposure and room for improvement through better privacy practices.

### Company Exposure Ranking

| Company   | Exposure Score |
| --------- | -------------- |
| Google    | 92/100         |
| Meta      | 81/100         |
| ByteDance | 68/100         |
| Amazon    | 60/100         |
| Snap Inc. | 54/100         |
| Apple     | 25/100         |

### Key Risks Identified

* Ad Tracking
* Social Exposure
* Location Tracking
* Behavioral Profiling
* Cross-platform Data Correlation

### Most Valuable Data Assets

1. Social Graph
2. Behavioral / Attention Data
3. Location History
4. Payment & Spending Signals
5. Photos / Visual Identity
6. Search Intent History

### Privacy Improvement Recommendations

* Disable unnecessary location tracking
* Restrict app location permissions
* Review third-party app access
* Enable two-factor authentication (2FA)
* Limit profile visibility on social platforms

## What I Learned

* Digital services collect different categories of personal information.
* Cross-platform data can create detailed user profiles.
* Privacy scores can improve significantly with simple security measures.
* Data visualization makes privacy risks easier to understand.
* AI can generate complex dashboards quickly from structured prompts.
