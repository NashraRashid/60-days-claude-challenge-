# Day 36 – Cognitive Pattern Explorer

## Project

Cognitive Pattern Explorer

## Objective

Build an interactive web application that helps users explore and understand their thinking patterns through real-life scenarios, reflection activities, and personalized insights.

---

## Features

- Calm Mode and Stress Mode
- Everyday Scenario Challenges
- Drag-and-Drop Activities
- Thinking Sequence Exercises
- Personalized Reflection Journal
- Thinking Profile Comparison
- Responsive User Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI

---

## What I Learned

- How thinking patterns change under calm and stressful situations.
- The importance of reflection in improving decision-making.
- How interactive activities can help users better understand cognitive behavior.
- Building responsive web applications using HTML, CSS, JavaScript, and Claude AI.

---

## Challenges Faced

- Designing meaningful scenario-based interactions.
- Managing user responses across multiple chapters.
- Creating a simple and engaging reflection experience.

---

## Outcome

Successfully developed a Cognitive Pattern Explorer that guides users through interactive activities, helping them analyze their thinking style and reflect on their decision-making process in different situations.

---


## Key Takeaway

This project demonstrated how AI-powered interactive applications can encourage self-reflection and help users better understand their cognitive patterns through engaging activities and personalized feedback.
